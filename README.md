# Dynamic Brain Motion Simulation Platform

This repository contains the CAD models, technical drawings, and firmware for a pneumatic-based platform designed to simulate physiological brain micromotion for neurosurgical applications.

## Contents

- CAD/ → 3D models (STEP and Fusion files)
- drawings/ → Technical engineering drawings (PDF)
- firmware/ → Arduino control code
- images/ → System renders and assembly views

## Purpose

The system replicates cardiac-induced brain tissue pulsation (4–150 µm) using a Scotch yoke-driven pneumatic actuation system.
